# General info
App for classes: Reliability and diagnostics of digital circuits

## Main parts of the project
* Bit generator
* Coder
* Transmission channel
* Decoder

## Used protocols
* Stop-And-Wait
* Go-Back-N
* Selective-Repeat

## Used codes
* Parity-Code
* Mirroring-Code
* CRC32
* SHA256

## Programing language
* Python
* Makefile

## How to run
* python3 main.py
* make
